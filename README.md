`<F5>` : 构建项目
`<F6>` : make 当前目录下 build 文件夹

`q`/`Q` : 退出/保存并退出

`gd` : 跳转定义
`gD` : 跳转声明
`gi` : 跳转实现
`gr` : 查找引用
`K` : 悬浮文档
`[d` : 上一个诊断
`]d` : 下一个诊断

`<C-j>` : 跳转到下一个关键词 
`<C-s>` : 保存
`<C-i>`, `<C-o>` : 左右切换buffer
`<C-\>` : 删除当前 buffer
`<C-p>` : 打开文件树
`<C-u>` : 呼出/关闭终端

`<C-w>` + `w`/`h`/`j`/`k`/`l` : 切换窗口
